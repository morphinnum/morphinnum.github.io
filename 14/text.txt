Node.js is an open-source, cross-platform JavaScript run-time environmentfor executing JavaScript code server-side. Historically,JavaScript was used primarily for client-side scripting, in which scriptswritten in JavaScript are embedded in a webpages HTML, to be run client-sideby a JavaScript engine in the users web browser.
Node.js enables JavaScript to be used for server-side scripting,and runs scripts server-side to produce dynamic web page content beforethe page is sent to the users web browser.
